About creators

The main researchers involved in producing the data, or the authors of the publication, in priority order.

May be a corporate/institutional or personal name.

Name
The name of the creator
Examples: Smith,John; Miller, Elizabeth
Identifier
Uniquely identifies an individual or legal entity, according to various schemes.
The format is dependent upon scheme.
Identifier Scheme
The name of the name identifier scheme.
If Identifier is used, Identifier Scheme is mandatory. Examples: ORCID12, ISNI13
Scheme URI
The URI ofthe name identifier scheme.
Examples: http://www.isni.org or http://www.orcid.org

About titles

A name or title by which a resource is known.

About subjects

Subject, keyword, classification code, or key phrase describing the resource.

Scheme
The name of the subject scheme or classification code or authority if one is used.
Scheme URI
The Scheme URI of the subject identifier scheme.
Examples: http://id.loc.gov/authorities/subjects; http://dewey.info/

About contributors

The institution or person responsible for collecting, managing, distributing, or otherwise contributing to the development of the resource.

Name
The name of the contributor.
If Contributor is used, then Contributor Name is mandatory. Examples: Patel, Emily; Doe, John

The personal name format may be: family, given. Non‐roman names should be transliterated according to the ALA‐LC schemes.
Name Identifier
Uniquely identifies an individual or legal entity, according to various schemes.
The format is dependent upon scheme.
Name Identifier Scheme
The name of the name identifier scheme.
If Name Identifier is used, Name Identifier Scheme is mandatory. Examples: ORCID15, ISNI16
Scheme URI
The URI of the name identifier scheme.
Examples: http://www.isni.org, http://www.orcid.org
Contributor Type
The type of contributor of the resource.

About dates

Different dates relevant to the work.

YYYY,YYYY‐MM‐DD, YYYY‐MM‐DDThh:mm:ssTZD or any other format or level of granularity described in W3CDTF. Use RKMS‐ISO8601 standard for depicting date ranges. Example: 2004‐03‐02/2005‐06‐02

Date Type
The type of date.
If Date is used, Date Type is mandatory.

About alternate identifiers

An identifier or identifiers other than the primary Identifier applied to the resource being registered. This may be any alphanumeric string which is unique within its domain of issue. May be used for local identifiers. Alternate Identifier should be used for another identifier of the same instance (same location,same file).

Example, A local accession number: E‐GEOD‐34814

Identifier Type
The type of the Alternate Identifier. If AlternateIdentifier is used, Identifier Type is mandatory.
