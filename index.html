EZID (easy-eye-dee) makes it easy to create and manage unique, long-term identifiers.

  • You can create identifiers for any kind of entity--physical, digital, abstract, etc. For digital object identifiers, you can use EZID to maintain their current locations so any citation links you've created are never broken.
  • You can also store citation metadata with identifiers to help others discover your objects and help you maintain the identifiers. Several citation formats are possible.
  • EZID also provides you with a QR code for every identifier. This comes in handy for using in poster sessions, promotional materials about your research, and anywhere else you don't have space to reference your object fully.
  • EZID supports a number of persistent identifier technologies, including ARKs (Archival Resource Keys), and DataCite DOIs (Digital Object Identifiers).

EZID also has a programming interface. You need a login account for full use of EZID, but you can look up any existing identifier without an account. And, you can start a Free Trial today! Once you've seen how it works, please contact us for more information about getting an account.
